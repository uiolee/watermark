## 改动

- 默认为 Jpeg 格式，增加 png、webp 格式。
- 添加文本角度调整功能。
- 可选即时预览。
- 显示各参数的数值。
- 使用 `<img>` 实现长按或右键保存。隐藏 `<canvas>`。
- 增加原图预览。
- 显示图片大小、类型。
- 显示耗时。
- 使用 [picocss](https://github.com/picocss/pico) 美化界面。
